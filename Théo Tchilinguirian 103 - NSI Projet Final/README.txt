Ne pas sortir les fichiers de l'archive

D'abord, qu'est-ce qu'une case libre?
	Une case libre, c'est une case qui n'a pas été retirée, et sur laquelle il n'y a pas de pion.

Règles du jeu :

	Règle de base:

		Lorsque c'est votre tour:
			Choisissez une des huit cases autour de votre pion pour avancer sur cette case; la case doit être libre
			Choisissez une case libre à supprimer partout sur le plateau
	
	Règle secondaire:
		
		Lorsque c'est votre tour:
			Choisissez une des huit cases autour de votre pion pour avancer sur cette case; la case doit être libre
			La dernière case sur laquelle vous vous trouviez est maintenant supprimée


But du jeu:
	
	Enlever les huit cases autour de votre adversaire


Options facultatives à choisir en début de partie:
	
	En début de partie, vous pouvez choisir parmi 3 options facultatives:
		
		1. Position aléatoire des joueurs (par défaut: désactivé)  -> Les joueurs sont placés aléatoirement sur le plateau en début de partie
		2. Jouer avec les règles de base (par défaut: activé)  -> Vous jouez avec les règles de base du jeu
           	3. Permission de sauter son tour (par défaut: désactivé)  -> Vous permet de sauter votre tour en appuyant sur 'Entrée' au moment de choisir une case où se déplacer
